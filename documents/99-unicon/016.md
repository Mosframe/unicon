# 클라이언트 기본 클래스 구성
## 목표 : 가능한 Unity3D와 비숫한 구성으로 제작한다.

## Scene 구조
```
    Scene
        ├ GameObject (1)
        │    ├ GameObject (1-1)
        │    └ GameObject (1-2)
        │        ├ GameObject (1-2-1)
        │        └ GameObject (1-2-2)
        ├ GameObject (2)
        │    ├ GameObject (2-1)
        ...
```
## GameObject 구조
```
    // GL.Object3D에서 상속받은 모든 오브젝트들을 GameObject라 칭한다.

    // 인스턴스 생성 ( 프리팹 )
    // 컴포넌트 관리
    // 테그 관리
    // 레이어 관리

    GameObject : GL.Object3D
        ├ GameObject ( core = this )
        └ Components
            ├ Component
            ├ Transform
                ├ Camera : Component
                │   └ core : GL.Camera
                ├ Light : Component
                │   └ core : GL.Light

    // Transform은 한개만 존재해야 한다.
    ...

```
## Transform 구조
```
    // 트랜스폼 관리
    // 부모/자식들들 관리

    Transform : Component
        └ core : GL.Object3D
```
